# Welcome to NostalgiaOS!

Thank you for exploring this retro operating system experience.

## What is NostalgiaOS?

NostalgiaOS is a browser-based operating system that brings back the nostalgic feel of classic computing interfaces. It features:

- **File Management**: Create, edit, and organize files and folders
- **Applications**: Calculator, Media Player, Watercolour drawing app, and more
- **Games**: Solitaire, Chess, Bombbroomer, Pong, Snake, and Happy Turd
- **Customization**: Personalize your desktop background and settings

## Getting Started

1. Explore the **Start Menu** to launch applications
2. Use **My Computer** to browse your file system
3. Create new files and folders using the context menu
4. Try the **Watercolour** app to create digital art
5. Check out the **Media Player** to enjoy music

## Tips

- Right-click on the desktop or in file explorer windows to access context menus
- Drag windows around to organize your workspace
- Your data is saved locally in your browser
- Check the **Storage Manager** to see how much space you're using

Enjoy your retro computing experience!
